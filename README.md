# CHUCAR
## 딜러와 사용자가 중고차 매매를 자유롭게 할 수 있도록 만든 소셜 어플리케이션
2개월 개발, 50% 기여 (2인 프로젝트)

# 백엔
* 서버 제작 및 배포
* 데이터베이스 구축

# 프론트엔드
* 어플리케이션 제작
* UI / UX 디자인

위 부분에서 백엔드를 담당했습니다.

# 주요기능

* 로그인 / 회원가입
* 결제
* 견적서 요청 / 견적서 송신
* 보낸 견적서 리스트 확인, 검색
